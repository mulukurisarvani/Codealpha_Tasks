# CodeAlpha - Flashcard Quiz App

An interactive, responsive 3D Flashcard Quiz web application built using pure vanilla HTML, CSS, and JavaScript. This platform provides complete client-side data persistence (CRUD) allowing users to dynamically customize their flashcard decks for real-time studying. Developed during my **Web & App Development Internship** at **CodeAlpha**.

---

## 🚀 Features
* **3D Flip Transitions:** Built using unique CSS transform properties to recreate realistic card-flipping mechanics when selected.
* **Linear Array Navigation:** Seamless "Next" and "Previous" pagination logic using mathematical loop counters.
* **Full CRUD Customization:** Users can write custom cards, modify or edit active question objects, or purge data rows cleanly via interactive control forms.
* **LocalStorage State Caching:** Saves any custom cards directly to the browser's native engine, preventing database loss on sudden tab closure or context refreshes.

---

## 🛠️ Built With
* **HTML5:** Modular frameworks structure layout setups for interactive quiz sections and control tracking nodes.
* **CSS3:** Custom styling grids managed over responsive dimensions and 3D perspectives variables.
* **JavaScript (ES6+):** Complete lifecycle event observation, local caching JSON formatting serialization workflows, and manual component styling manipulations.
